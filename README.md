# Mini Relational Query Engine
A simple relational query engine built in java to demostrate query execution, indexing, optimization, caching and scheduling.

## Features
- CSV-backed tables
- SQL-style querying
- Query execution operators
- Query optimization
- Indexing
- LRU caching 
- Query scheduling
- Performance benchmarking
