# Mini Relational Query Engine

## Project Goal
Build a relational query engine that:
- Loads CSV tables
- Executes SQL queries
- Demonstrates indexing, query optimization, caching and scheduling

## Architecture
CSV Files -> Storage layer -> Query engine -> Results & metrics

## Initial Features
- CSV loading
- Table scan
- Filtering
- Projection

## Future Features
- Join operations
- Hash indexing
- Query optimizer
- LRU cache
- Query scheduler
- Benchmarking
